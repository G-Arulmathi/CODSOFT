# Titanic Survival Prediction

## Overview

This project is part of the CodSoft Data Science Internship.

The objective is to predict whether a passenger survived the Titanic disaster using machine learning techniques.

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn

## Machine Learning Algorithm

Random Forest Classifier

## Features Used

* Passenger Class (Pclass)
* Gender (Sex)
* Age
* Siblings/Spouses Aboard (SibSp)
* Parents/Children Aboard (Parch)
* Fare
* Embarked

## Steps Performed

1. Data Cleaning
2. Handling Missing Values
3. Label Encoding
4. Train-Test Split
5. Model Training
6. Prediction and Evaluation

## Results

The model achieved approximately 82% accuracy on the test dataset.

## Conclusion

The Random Forest model successfully predicted passenger survival using demographic and travel information.
